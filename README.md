Project Proposal: Analyzing and Predicting Flight Delays Using Bureau of Transportation Statistics Data
- This project analyzes and predicts U.S. airline flight delays using the Bureau of Transportation Statistics (BTS) On-Time Performance dataset. 
- The data include around 70,000 flights from July 2025 with fields such as airline, airport, date, scheduled and actual times, delay durations, and causes (weather, carrier, system, and late aircraft).
